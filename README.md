# Papyrus

Business management software for small business owners.

## About
Papyrus is a cross-platform Flutter application designed to help small business owners manage their operations efficiently. It provides tools for sales, purchases, inventory, wallets, and team management.

## Features
- **Dashboard**: High-level overview of business metrics and low stock alerts.
- **Sales & Purchases**: Create and manage invoices for customers and suppliers.
- **Inventory Management**: Track products, stock levels, and categories.
- **Ledger (Len Den)**: Manage debts and loans with parties.
- **Income/Expense (Ay Bay)**: Detailed tracking of cash flow.
- **Wallets**: Multi-wallet support with fund transfer capabilities.
- **Team Management**: Role-based access control and member invitations.
- **Settings**: Customizable invoice prefixes, currency, and VAT.

## Tech Stack
- **Flutter**: Cross-platform UI framework.
- **Supabase**: Backend for authentication and real-time database.
- **GoRouter**: Advanced routing and navigation.
- **Provider**: State management.
- **Lucide Icons**: Modern and consistent iconography.

## Getting Started
1. Clone the repository.
2. Run `flutter pub get`.
3. Configure your Supabase credentials in `lib/main.dart`.
4. Run `flutter run`.
